Rock-Paper-Scissors game

In this game, users can select "Rock," "Paper," or "Scissors" to play against a computer opponent. JavaScript handles game logic, determining wins, losses, or ties based on set rules. Results are displayed, and a scoring system tracks the player's performance, with an option to reset the score. An "Auto Play" feature allows automated rounds.
Tools and Technologies used: HTML for structure, CSS for styling, and JavaScript for interactivity.

Simple Calculator

This web calculator offers basic arithmetic functionality with numeric buttons (0-9), decimal point (.), and operators (+, -, *, /). As users input values, the current calculation is displayed in real-time. Clicking the equals (=) button evaluates and displays the result. A "Clear" button resets the calculation. It also includes local storage to save the current calculation
